{\bfseries{open\+MTPK}} is an open-\/source (intended) mathematics package written in C++ with a primary focus on Numbery Theory and Cryptographic algorithms, Linear Algebra, and Machine/\+Deep learning concepts as well as a range of language API\textquotesingle{}s for integration with pre-\/existing projects. open\+MTPK is originally inspired from undergraduate coursework and work done on \href{https://github.com/akielaries/vpaSTRM}{\texttt{ vpa\+STRM}}. The project welcomes contributors and is in need of mathematics focused/interested programmers comfortable in C and C++. If interested see \href{https://github.com/akielaries/openMTPK/blob/main/CONTRIBUTING.md}{\texttt{ here}}!

The goal is to make a reusable mathematics library allowing users to call within their own projects. Some of the implementations seen in the package were first prototypes in Wolfram Mathetmatica, then translated to C++ for the package. Look in the \href{https://github.com/akielaries/openMTPK/tree/main/samples}{\texttt{ samples}} folder for examples on using these tools in your own project.\hypertarget{md_README_autotoc_md1}{}\doxysubsection{API Bindings}\label{md_README_autotoc_md1}

 \hypertarget{md_README_autotoc_md6}{}\doxysubsection{Build Bindings}\label{md_README_autotoc_md6}
The binding process leverages the use of Swig, specifically the fork authored by {\itshape sethrj} that makes use of the Fortran binding process. See \href{https://github.com/swig-fortran/swig}{\texttt{ here}}. Each API comes with a custom Makefile for compiling a wrapper for the respective language, but does not take care of storing files in necessary filepaths needed by the compiler/interpreter. \hypertarget{md_README_autotoc_md7}{}\doxysubsubsection{Install Swig}\label{md_README_autotoc_md7}

\begin{DoxyCode}{0}
\DoxyCodeLine{\# clone the fork of Swig}
\DoxyCodeLine{\$ git clone git@github.com:swig-\/fortran/swig.git}
\DoxyCodeLine{\$ cd swig/}
\DoxyCodeLine{\# run autotools}
\DoxyCodeLine{\$ ./autogen.sh}
\DoxyCodeLine{\# install}
\DoxyCodeLine{\$ make}
\DoxyCodeLine{\$ make check}
\DoxyCodeLine{\$ make install}

\end{DoxyCode}
